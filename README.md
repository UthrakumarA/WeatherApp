# Weather Information Website 🌦️

A responsive Weather Information Website built using React.js that allows users to search for any city and view real-time weather information.

## Features

- Search weather by city name
- Display temperature, humidity, and wind speed
- Dynamic background image based on searched city
- Real-time weather data using OpenWeather API
- Responsive user interface
- Error handling for invalid city names

## Technologies Used

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- OpenWeather API
- Unsplash API
